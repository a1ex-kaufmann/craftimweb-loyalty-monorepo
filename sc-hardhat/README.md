# loyalty cotracts

Smart Contracts for loyalty system

- Language: Solidity v0.8.20

- Project framework: hardhat (ethers) + typechain

- Nodejs: v18.17.1

## Installation & Usage

1. Install packages
```
yarn
```

2. Build project
```
yarn build
```

### Testing

```
yarn test
```

### Run linter

For .sol files
```
yarn lint:sol
```

For .ts files
```
yarn lint:ts
```

### Run prettier

For .sol files
```
yarn prettier:sol
```

For .ts files
```
yarn prettier:ts
```
